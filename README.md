An interactive web-based car driving game built with HTML5 Canvas and JavaScript.

## Features

- Smooth car controls with keyboard arrows
- Real-time speed and distance tracking
- Animated road with moving lane markers
- Mobile-friendly touch controls
- Responsive design

## Controls

- **Arrow Up** - Accelerate
- **Arrow Down** - Reverse/Brake
- **Arrow Left** - Turn left
- **Arrow Right** - Turn right
- **Space** - Brake

## How to Play

1. Visit the game: [Play Now](https://syedabersabil.github.io/web-car-drive-game/)
2. Use arrow keys to control your car
3. Try to drive as far as possible!

## Technologies Used

- HTML5 Canvas
- CSS3 (Gradient backgrounds, flexbox)
- Vanilla JavaScript (Game loop, physics)

## Local Setup

```bash
git clone https://github.com/syedabersabil/web-car-drive-game.git
cd web-car-drive-game
```

Open `index.html` in your browser to play!

## Deployment

This game is deployed on GitHub Pages. To deploy your own version:

1. Go to repository Settings
2. Navigate to Pages section
3. Select main branch as source
4. Save and wait for deployment
